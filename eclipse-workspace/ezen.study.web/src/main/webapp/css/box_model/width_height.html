width 속성

box-sizing 속성은 CSS의 테두리 영역의 크기를 결정한다.

content-box: 지정한 CSS width 및 height를 컨텐츠 영역에만 적용한다. border, padding, margin은 따로 계산되어 전체 영역이 설정값보다 커질 수 있다.

border-box: 지정한 CSS width 및 height를 전체 영역에 적용한다. border, padding, margin을 모두 합하기 때문에 컨텐츠 영억이 설정값보다 작아질 수 있다.

content-box 200px

border-box 200px

content-box 50%px

border-box 50%px

height는 요소의 높이를 지정하는 속성이다. height1
