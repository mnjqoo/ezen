overflow 속성

overflow는 컨텐츠가 요소의 컨텐츠 영역을 벗어나는 경우 처리방법을 지정하는 속성이다. overflow 속성을 방향별로 지정하기 위한 상세속성 overflow-X, overflow-Y를 사용할 수 있다.
