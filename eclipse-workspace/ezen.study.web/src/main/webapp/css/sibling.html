General Sibling Combinator(기본 형제 선택자)

선행 선택자 뒤의 후행 선택자 선택한다.

기본 형제 선택자는 선행 선택자와 후행선택자를 "+"로 구분하여 선언합니다.

기본 형제 선택자는 선행 선택자와 후행선택자를 "+"로 구분하여 선언합니다.

Adjacent Sibling Combinator(인접 형제 선택자)

선행 선택자 뒤에 인접하여 등장하는 모든 후행 선택자를 선택한다.

인접 형제 선택자는 선행 선택자와 후행선택자를 "~"로 구분하여 선언합니다.

인접 형제 선택자는 선행 선택자와 후행선택자를 "~"로 구분하여 선언합니다.
